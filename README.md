# List Data type features
- enclosed in square brackets.
- Python lists may have objects of different data types.
- A Python list is mutable
- One or more objects from the list can be removed or added.
- A list may have same item at more than one index positions.

# Tuple Data type features
- enclosed in parentheses ().
- tuple may have objects of different data types.
- tuple is immutable.
- cannot be modified, removed or added.
